SPLASH SCREEN (v2 - uses your actual logo image, simpler animation)

This version is simpler than the first one: just a soft glow fade-in behind
your logo, plus the logo itself scaling up with a bouncy spring. No separate
animated bars -- it uses your real logo.png file directly, so it always
matches exactly.

STEP 1 -- Insert this as the VERY FIRST thing inside <body>, before
<div class="mobile-menu">:

<div id="splash-screen">
  <div class="splash-glow"></div>
  <div class="splash-logo">
    <img src="assets/images/logo.png" alt="Dopamine"/>
  </div>
</div>

STEP 2 -- Add this CSS link in <head>, with your other CSS links:
<link rel="stylesheet" href="css/splash.css"/>

STEP 3 -- Add this script tag near your other script tags (works fine
anywhere since it just waits for DOMContentLoaded):
<script src="js/splash.js"></script>

No font changes needed this time since it's an image, not text.
